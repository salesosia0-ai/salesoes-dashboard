# Guiones de llamada

En esta carpeta están los guiones de llamada comerciales por sector.

Cada guion sigue una estructura similar:
- Apertura y contexto de la llamada.
- Preguntas de descubrimiento.
- Presentación de valor adaptada al sector.
- Cierre con siguiente paso claro (demo, reunión, etc.).[web:71][web:72]

## Archivos

- `guion_asesorias.md`: guión para asesorías.
- `guion_clinicas.md`: guión para clínicas dentales.
